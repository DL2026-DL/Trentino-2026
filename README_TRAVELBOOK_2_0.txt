TravelBook 2.0 - Meteo live + PWA

Modifiche incluse:
- Meteo live da Open-Meteo per Tesero.
- Card "Meteo Tesero ora" con temperatura, percepita, umidità, vento e pioggia.
- Previsioni a 7 giorni.
- Pulsante "Aggiorna meteo".
- Cache offline per l'app, ma richiesta meteo sempre da rete.

Come aggiornare GitHub Pages:
1. Apri il repository GitHub.
2. Carica/sostituisci index.html, manifest.json e service-worker.js.
3. Carica la cartella assets se GitHub te la chiede.
4. Commit changes.
5. Apri il sito e fai refresh completo.

Su iPhone, se vedi ancora il vecchio meteo:
- Apri Safari > sito TravelBook.
- Aggiorna la pagina.
- Se l'hai aggiunta alla Home, chiudi e riapri l'app.
- In caso di cache persistente, elimina l'icona dalla Home e aggiungila di nuovo.
